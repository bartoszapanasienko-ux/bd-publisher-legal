Privacy Policy Terms of Service

This is an internal, single-user tool used exclusively by the operator to publish podcast clips to their own TikTok account. It does not collect, store, or process data from any third-party users.

1. Overview

Bratnie Dusze Publisher ("the Tool") is a private, server-side automation tool operated by Bartosz [Last Name] for the purpose of scheduling and publishing short-form video content from the Bratnie Dusze podcast to a single TikTok account owned and controlled by the same operator.

The Tool is not a consumer application. It has no user-facing interface accessible to the public and does not serve, register, or interact with any users other than the operator.

2. Data We Collect

The Tool processes the following data, solely on the operator's local machine:

— Video files exported from Opus.pro belonging to the operator's own account
— Metadata associated with those videos (title, transcript text, duration)
— TikTok OAuth tokens issued to the operator's own TikTok account
— A local SQLite database storing the publication queue

No data is collected from any third party. No user accounts are created. No personal data of any external person is processed.

3. How Data Is Used

Data is used exclusively to:

— Generate post descriptions using an AI language model (Anthropic Claude API)
— Upload video content to the operator's TikTok account via the TikTok Content Posting API
— Schedule and track publication status locally

4. Data Storage

All data is stored locally on the operator's personal computer. No cloud storage, third-party databases, or external servers are used beyond the APIs listed above (TikTok, Anthropic). No data is shared with any other party.

5. Third-Party Services

The Tool integrates with:

— TikTok Content Posting API — to publish video content. Governed by TikTok's own Privacy Policy and Terms of Service.
— Anthropic Claude API — to generate post descriptions. Governed by Anthropic's Privacy Policy.
— Opus.pro — to retrieve video exports from the operator's own account.

6. Data Retention

Data is retained locally for as long as the operator chooses. The operator may delete all data at any time by removing the local database file.

7. Your Rights

As this Tool processes no data belonging to third parties, there are no third-party rights to exercise. The operator retains full control over all data processed by the Tool.
